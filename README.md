This repository contains the required tasks for *Assignment 2, focusing on basic process creation using fork(), simple C programs, and building projects using a **Makefile* on Ubuntu Linux.

A detailed explanation of each example in answers.txt
Explanation of the roles of the *Linker* and *Loader*
A working *Makefile* to compile all programs
A MIT License
